# Features
This article is about AdGuard for Mac, a multifunctional ad blocker that protects your device at the system level. To see how it works, [download the AdGuard app](https://agrd.io/download-kb-adblock)
## Main window[​](https://adguard.com/kb/adguard-for-mac/features/#main-window "Direct link to heading")
Main screen with statistics on blocked ads and trackers and access to main settings
[Main window](https://adguard.com/kb/adguard-for-mac/features/main/)
## General[​](https://adguard.com/kb/adguard-for-mac/features/#general "Direct link to heading")
Basic settings to block search ads and website self-promotional ads, enable language-specific filters, and manage _Allowlist_
[General](https://adguard.com/kb/adguard-for-mac/features/general/)
## Filters[​](https://adguard.com/kb/adguard-for-mac/features/#filters "Direct link to heading")
Filters and user rules to fine-tune ad blocking
[Filters](https://adguard.com/kb/adguard-for-mac/features/filters/)
## DNS[​](https://adguard.com/kb/adguard-for-mac/features/#dns "Direct link to heading")
DNS servers and DNS filters that let you block ads, trackers, and dangerous websites
[DNS](https://adguard.com/kb/adguard-for-mac/features/dns/)
## Stealth Mode[​](https://adguard.com/kb/adguard-for-mac/features/#stealth-mode "Direct link to heading")
Settings to block cookies, UTM tags, and online analytics that collect user information
[Stealth Mode](https://adguard.com/kb/adguard-for-mac/features/stealth/)
## Security[​](https://adguard.com/kb/adguard-for-mac/features/#security "Direct link to heading")
A setting that blocks access to malicious and phishing websites and protects you from online threats
[Security](https://adguard.com/kb/adguard-for-mac/features/security/)
## Browser Assistant[​](https://adguard.com/kb/adguard-for-mac/features/#browser-assistant "Direct link to heading")
An extension that complements the desktop program and allows you to manage filtering from your browser
[Browser Assistant](https://adguard.com/kb/adguard-for-mac/features/browser-assistant/)
## Extensions[​](https://adguard.com/kb/adguard-for-mac/features/#extensions "Direct link to heading")
A feature that allows you to extend the functionality of websites by adding userscripts and userstyles
[Extensions](https://adguard.com/kb/adguard-for-mac/features/extensions/)
## Network[​](https://adguard.com/kb/adguard-for-mac/features/#network "Direct link to heading")
Additional network-related options and proxy settings
[Network](https://adguard.com/kb/adguard-for-mac/features/network/)
